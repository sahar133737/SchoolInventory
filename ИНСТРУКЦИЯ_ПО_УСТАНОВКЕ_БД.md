# Инструкция по установке базы данных

## Быстрый старт

### Способ 1: PowerShell (рекомендуется)

1. Откройте PowerShell от имени администратора
2. Перейдите в папку `WindowsFormsApp1\Database`
3. Выполните команду:
```powershell
.\Install-Database.ps1 -ServerInstance "localhost"
```

### Способ 2: SQL Server Management Studio

1. Откройте SQL Server Management Studio
2. Подключитесь к вашему SQL Server
3. Откройте и выполните файл `WindowsFormsApp1\Database\CreateDatabase.sql`
4. Откройте и выполните файл `WindowsFormsApp1\Database\SeedData.sql`

### Способ 3: Командная строка (sqlcmd)

```cmd
cd WindowsFormsApp1\Database
sqlcmd -S localhost -i CreateDatabase.sql
sqlcmd -S localhost -i SeedData.sql
```

## Что будет создано

- База данных **SchoolInventoryDB**
- 5 таблиц с индексами и связями
- Тестовые данные:
  - 10 категорий
  - 15 кабинетов
  - 15 ответственных лиц
  - 4 пользователя
  - 25 записей инвентаря

## Настройка подключения

После создания БД настройте строку подключения в `WindowsFormsApp1\App.config`:

```xml
<connectionStrings>
    <add name="DefaultConnection" 
         connectionString="Server=localhost;Database=SchoolInventoryDB;Integrated Security=true;" />
</connectionStrings>
```

Подробные инструкции см. в файле `WindowsFormsApp1\Database\README.md`
